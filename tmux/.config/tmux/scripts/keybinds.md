# tmux keybinds

> Prefix: `Ctrl+A`

## Navigation

| Key | Action |
|-----|--------|
| `prefix + h/j/k/l` | Move between panes (vim-style) |
| `Alt+Shift+H/J/K/L` | Resize pane |

## Windows & Panes

| Key | Action |
|-----|--------|
| `prefix + \|` | Split horizontally (current path) |
| `prefix + -` | Split vertically (current path) |
| `prefix + x` | Kill pane (no confirm) |
| `prefix + z` | Zoom pane |

## Sessions

| Key | Action |
|-----|--------|
| `prefix + N` | New session |
| `prefix + s` | Fuzzy session switcher (sesh) |
| `prefix + w` | New git worktree popup |

## Copy Mode

| Key | Action |
|-----|--------|
| `prefix + [` | Enter copy mode |
| `v` | Begin selection |
| `Ctrl+v` | Rectangle selection |
| `y` | Yank and exit |

## Misc

| Key | Action |
|-----|--------|
| `prefix + r` | Reload config |
| `prefix + ?` | Show this help |
