# Mac Planck Layout

### Layer 0 — Base

```
┌────────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┬────────┐
│ Tab    │ q    │ w    │ e    │ r    │ t    │ y    │ u    │ i    │ o    │ p    │ Bksp   │
├────────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼────────┤
│ Esc    │ a    │ s    │ d    │ f    │ g    │ h    │ j    │ k    │ l    │ ; :  │ ' "    │
├────────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼────────┤
│ LShift │ z    │ x    │ c    │ v    │ b    │ n    │ m    │ , <  │ . >  │ / ?  │ Enter  │
├────────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼────────┤
│ LAlt   │ LCtl │ LAlt │ Vol- │ Vol+ │ LAlt │ Spc  │ Raise│ M3   │ LCmd │ Lower│ Spc    │
└────────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┴────────┘
```

- `Lower` (FN_MO13): Hold to activate Layer 1
- `Raise` (FN_MO23): Hold to activate Layer 2
- `Lower + Raise`: Activates Layer 3
- `M3`: Macro | tmux prefix | `Ctrl+A`

### Layer 1 — Symbols & Numbers

```
┌──────┬──────┬────────┬────────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┐
│ ` ~  │  —   │ M2     │ M4     │  —   │ [ {  │ ] }  │ 7 &  │ 8 *  │ 9 (  │ - _  │  ▼   │
├──────┼──────┼────────┼────────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┤
│  ▼   │  —   │ M0     │  —     │ M1   │ {    │ }    │ 4 $  │ 5 %  │ 6 ^  │ = +  │ \ |  │
├──────┼──────┼────────┼────────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┤
│  ▼   │ M5   │  —     │  —     │  —   │ (    │ )    │ 1 !  │ 2 @  │ 3 #  │ .    │  ▼   │
├──────┼──────┼────────┼────────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┼──────┤
│  ▼   │  ▼   │  ▼     │ Bri-   │ Bri+ │  ▼   │  ▼   │  ▼   │ 0    │  ▼   │  ▼   │  ▼   │
└──────┴──────┴────────┴────────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┘
```

- `M0`: Macro | vim | save all | `Esc → :wa → Enter`
- `M1`: Macro | vim | search & replace | `Esc → :%s/`
- `M2`: Macro | vim | close buffer | `Esc → ␣bd`
- `M4`: Macro | macOS | emoji picker | `Ctrl+Cmd+Space`
- `M5`: Macro | macOS | zoom toggle | `Alt+Cmd+8`
- `▼`: Transparent (falls through to layer below)
- `—`: No key (KC_NO)
